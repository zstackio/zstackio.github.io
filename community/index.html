ZStack is an open source project provided under license Apache License 2.0. There are a couple of ways to ask questions regarding usage, development, and any other topics you may be interested in.

Discussion Groups

  Google Groups mailing list

Google Group is the primary place to ask questions and find answers.

  Stack Overflow

You can also ask questions on Stack Overflow with a tag zstack. Click to ask yours.

  GitHub

You can also submit questions as issues to GitHub. Please ensure your questions are labeled as otherwise we may not notice you are asking questions and may not be able to reply in time.

  IRC Channel

You can find us on Freenode IRC with channel number #zstack.

Source Code

Check out source code in our GitHub repos:

  • zstack: the core orchestration.
  • zstack-utility: the repo containing various agents(e.g. kvmagent, virtual router VM agent) and tools(e.g. zstack-cli, zstack-ctl) in Python.
  • zstack-dashboard: the web UI.
  • zstack-woodpecker: the automated testing framework.
